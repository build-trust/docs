<RequestExample>

```python main.py
from ockam import Agent, Model, Node, Repl
from sys import argv


async def main(node):
    agent = await Agent.start(
        node=node,
        instructions='''
            You are an agent that specializes
            in giving advice on creating safe,
            secure, and scalable software.
        '''
    )

    await Repl.run(agent, argv[1])


Node.run(main)
```
