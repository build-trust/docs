<CodeGroup>

```python images/main/main.py
from ockam import Agent, Node, Repl
from sys import argv


async def main(node):
    agent = await Agent.start(node, "You are Jack Sparrow.")
    await Repl.start(agent, argv[1])


Node.start(main)
```


```yaml ockam.yaml
name: hello
pods:
  - name: main-pod
    containers:
      - name: main
        image: main
        args: [localhost:9000]
    portals:
      outlets:
        - to: localhost:9000
```


```docker images/main/Dockerfile
FROM cgr.dev/chainguard/python:latest-dev AS dev

ENV PATH="/tmp/venv/bin:$PATH"
COPY requirements.txt requirements.txt
RUN python -m venv /tmp/venv && pip install -r requirements.txt

FROM cgr.dev/chainguard/python:latest

WORKDIR /app

COPY main.py main.py
COPY --from=dev /tmp/venv /app/venv
ENV PATH="/app/venv/bin:$PATH"

ENTRYPOINT ["python", "main.py"]
```
